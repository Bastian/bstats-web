# bStats Web

The frontend for [bStats](https://bstats.org), a free and open-source statistics
service for Minecraft plugins and mods.

## Running locally

Please refer to the [parent repository](https://github.com/Bastian/bStats) that
contains this repository as a submodule and instruction on how to run the entire
projects (including backend, database, etc.) locally.
